CLI-CALCULATOR

A simple command-line calculator.

You can execute the "build.sh" script to build the calculator :
chmod +x build.sh
sudo ./build.sh
Or use the make command :
make

Examples of utilisations :
tcalc 12+8
=> 20
tcalc 12+8*2
=> 40
tcalc 12+(8*2)
=> 28
tcalc 10.2*8
=> 81.600000
tcalc 7/2
=> 3
tcalc 7%2
=> 1
tcalc 7.0/2
=> 3.500000
